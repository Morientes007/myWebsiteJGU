Research Highlights

Explainable AI & Dynamic Programming

Developed methods for automatically generating explanations for computation results, particularly in dynamic programming algorithms. This work was conducted under DARPA's Explainable AI (XAI) program.

Message Passing Language (MPL)

Designed and implemented a type-safe concurrent programming language with a novel type system and abstract machine for execution.

Domain-Specific Languages

Created DSLs for various domains including game-theoretic matching (MatchMaker), explanatory decision making (MADMAX), and combinatorial optimization.

Incremental Computing

Current work on differential execution techniques for automatic incrementalization of computations, enabling efficient recomputation when inputs change.
